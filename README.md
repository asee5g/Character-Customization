# Character Customization (Unreal Engine 5)

A **modular character customization system** built in **Unreal Engine 5** using **C++ and Blueprints**.  
This project demonstrates dynamic mesh swapping, UI-driven customization, camera focus transitions, and persistent save/load functionality.

---

## 🎮 Features
- Modular customization: Hair, Face, Chest, Legs, Hands 
- Dynamic mesh swapping using DataTables 
- Smooth camera transitions to selected body parts  
- Character rotation inside the UI 
- Item selection UI with highlight indicators
-   Save & Load system using SaveGame + GameInstance
- Combination of **C++ gameplay code** and **Blueprint scripting**  

---

## 🧠 What I Learned
- Implementing **C++ systems** for modular skeletal mesh components  
- Creating reusable **UMG widgets** for item selection 
- Using **DataTables** to manage item IDs, meshes, and preview icons 
- Working with **Delegates** for clean UI -> Character communication
- Handling **TimeLine-Based** camera movement
- Building scalable architecture for expandable customization systems
- Implementing persistent data saving for character setups
